# Personal Finance Expense Analytics App

A complete, production-ready React application for tracking personal finances and expenses, built with Vite, React Router, Recharts, and Framer Motion.

## Features
- Dashboard with real-time financial summaries
- Transactions management (Add, Edit, Delete, Search, Filter, Sort)
- Budget tracking with visual progress
- Interactive Analytics with Pie, Bar, and Line charts
- Responsive, modern glassmorphism UI with Dark Mode

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
